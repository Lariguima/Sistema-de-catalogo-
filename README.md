Sobre o Projeto:

  O Xama Aí é uma aplicação web voltada para facilitar a conexão entre
  usuários e estabelecimentos próximos. A plataforma permite visualizar
  lojas no mapa, acessar seus catálogos e realizar pedidos de forma rápida
  e intuitiva.

Funcionalidades Principais:

Cadastro e Login de Usuário
  Permite criar uma conta e acessar o sistema com e-mail e senha.

Visualização de Catálogo de Lojas 
  Ao clicar em um estabelecimento, o usuário pode ver produtos, detalhes
  da loja e comentários.

Tecnologias Utilizadas:

  HTML5
  CSS3
  JavaScript
